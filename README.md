# BomberMan Revolution / 1Q Console Project
-----------------
### 게임인재원에서 진행한 1Q Mini Project입니다.
``` 

Console 창 기반으로 만들었고, 프로그래머 학생 2인 협업 프로젝트입니다.
네트워크를 달아보거나 멀티 스레드를 사용해보는 등의 개인적인 도전이 포함된 프로젝트입니다.

네트워크 관련 지식이 전무한 상태였기 때문에 공부하며 진행했고,
『게임 서버 프로그래밍 교과서』라는 책의 예제로 제공되는 소스코드를 분석하며 사용했습니다.

멀티 스레드도 Mutex를 거의 사용하지않고 역할을 분담해서 하는 정도로만 사용했습니다.

결과적으로 도전은 실패했지만 공부에는 도움이된 프로젝트였습니다.

세부적인 사항은 consoleGame1 폴더에 안내되어 있습니다.
```
